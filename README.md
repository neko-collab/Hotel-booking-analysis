Hotel Booking Analysis & Insights
Python Project on Hotel Reservations, Cancellations, and Pricing Trends
This repository presents a detailed analysis of hotel booking data using Python (Pandas & Matplotlib) to uncover patterns in reservations, cancellations, and pricing (Average Daily Rate, ADR). The project covers:
•	Comparing reservation status across hotel types and months.
•	Analyzing ADR trends over time and across canceled vs. completed bookings.
•	Visualizing customer demographics, top countries, and market segments.
•	Providing actionable insights to optimize hotel operations, pricing, and customer experience.
Data Source: Open-source hotel booking dataset (cleaned for analysis).
